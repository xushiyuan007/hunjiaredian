# 🏠 婚姻家事热点新闻智能推送系统

专为婚姻家事律师打造的**每日自动化热点新闻搜集工具**，助力自媒体内容创作。

## ✨ 功能特点

- 🕒 **定时自动执行**: 每天早上9点自动搜集热点新闻
- 🔍 **专业关键词**: 覆盖12类婚姻家事核心领域
- 📂 **智能分类**: 自动按法规解读、财产纠纷、子女抚养等分类整理
- 📢 **多渠道推送**: 支持钉钉、飞书群机器人推送
- 📝 **结构化输出**: Markdown格式报告，直接用于自媒体素材
- ☁️ **GitHub Actions部署**: 零服务器成本，永久免费运行

## 🚀 快速部署

### 第一步：创建GitHub仓库

1. 在GitHub创建一个新的私有仓库
2. 将本项目所有文件上传到仓库

### 第二步：配置推送Webhook（二选一或都配置）

#### 选项A：钉钉推送
1. 打开钉钉群 → 群设置 → 智能群助手 → 添加机器人
2. 选择「自定义」机器人
3. 安全设置选择「自定义关键词」，添加 `婚姻家事`
4. 复制Webhook地址

#### 选项B：飞书推送
1. 打开飞书群 → 群设置 → 群机器人 → 添加机器人
2. 选择「自定义机器人」
3. 复制Webhook地址

### 第三步：配置GitHub Secrets

1. 进入仓库 → Settings → Secrets and variables → Actions
2. 点击「New repository secret」
3. 添加以下Secret（按需添加）:

| Secret名称 | 内容 | 必填 |
|-----------|------|------|
| `DINGTALK_WEBHOOK` | 钉钉机器人Webhook地址 | 可选 |
| `LARK_WEBHOOK` | 飞书机器人Webhook地址 | 可选 |

### 第四步：启用Actions

1. 进入仓库 → Actions
2. 找到「婚姻家事每日热点新闻推送」工作流
3. 点击「Enable workflow」启用
4. 点击「Run workflow」可立即测试运行

## ⏰ 执行时间配置

默认每天 **北京时间9:00** 执行。如需修改时间，编辑 `.github/workflows/daily-news.yml` 中的cron表达式：

```yaml
schedule:
  - cron: '0 1 * * *'  # UTC时间，北京时间 = UTC + 8
```

| 北京时间 | UTC时间 | cron表达式 |
|---------|---------|-----------|
| 8:00 | 0:00 | `0 0 * * *` |
| 9:00 | 1:00 | `0 1 * * *` |
| 10:00 | 2:00 | `0 2 * * *` |

## 📞 常见问题

**Q: Actions运行失败怎么办？**
- 检查Secrets是否配置正确
- 查看Actions日志中的错误信息
- 确认Webhook地址是否有效

**Q: 可以自定义搜索关键词吗？**
- 可以！编辑 `src/news_fetcher.py` 中的 `keywords` 列表

**Q: 推送频率可以调整吗？**
- 可以，修改workflow中的cron表达式即可

**Q: 这个会产生费用吗？**
- 不会！GitHub Actions对公开仓库完全免费，私有仓库每月有2000分钟免费额度，完全够用
